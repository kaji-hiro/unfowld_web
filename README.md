  6.0.2.1
* Database
  local: MySQL
  production: postgreSQL

### 開発時実行コマンド

* Server start
  bundle exec rails s -b 'your IP Adress'
* Webpack development server start
  bin/webpack-dev-server
* Open rails console
  bundle exec rails c
* Running real-time tests
  bundle exec guard
